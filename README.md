# Automated Poker Game in Smalltalk

This is an automated poker game created in Smalltalk.

## Installation

1. Install Pharo from https://pharo.org/.
2. Clone the repository to your local machine.
3. Navigate to the project directory.

## Usage

Use the Pharo GUI to start the program and begin playing the game.

## Description

Program accepts a set of hands and calculates the winning hand.
